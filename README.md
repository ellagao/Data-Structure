  b. display polynomial  
  c. copy  
  d. summation  
  e. subtraction  
  f. multiplication  
  g. evaluation  
  h. destruction  
  i. emptying  
  j. differential  
  k. indefinite integral  
  l. definite integral  
  m. greatest common divisor(GCD)  
  n. lowest common multiple (LCM)  
  o. division  
  p. mod   
  q. power  
2. Four Arithmetic Operations Calculator
3. Binary Tree:  
  a. create  
  b. preorder traversal
